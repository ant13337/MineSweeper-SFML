# MineSweeper (SFML, C++)

This is a Minesweeper game written in C++ with SFML.

## Project Structure

- Source code: `CodePC/MainProgram`
- Executable: `Bin/x64`
- Best times file: `Bin/times.txt`

## Dependencies

This project uses SFML 2.5.0.

- SFML license file in this repo: `External/x64/SFML-2.5.0/license.md`
- Additional third-party notes: `THIRD_PARTY_LICENSES.md`
